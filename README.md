# Facebook Graph API Viewer

This project is a web application that allows users to fetch and view posts from a Facebook page using the Facebook Graph API. It demonstrates API integration, JavaScript logic, and responsive UI design.

# Features

•	Fetch posts from a specific Facebook page using an access token.

•	Search and filter posts by keywords.

•	Display post details such as message, date, and statistics in organized cards.

•	Loading indicators and error messages for better user experience.

•	Responsive design suitable for both desktop and mobile screens.

# Project Structure

facebook-graph/

│

├─ index.html          # Main HTML interface

├─ style.css           # CSS for layout, cards, buttons, and responsiveness

├─ script.js           # JavaScript for fetching Facebook posts, filtering, and rendering UI

└─ config.js           # Stores Facebook API access token


# Instructions to Run

1.	Open index.html inside the facebook-graph folder.

2.	The app will automatically open in your browser.
 
3.	Make sure to set a valid Facebook API access token in config.js.

No server setup is required; the project runs entirely in the browser.

# Technologies Used

•	HTML5, CSS3, JavaScript (ES6)

•	Facebook Graph API

•	Bootstrap Icons for UI elements

# Notes

•	Ensure your Facebook access token has the necessary permissions to read page posts.

•	Handle API keys and access tokens securely.

•	Check API rate limits to avoid errors during fetching.
